## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/AyushIos2005/agencies.git
```

Go to project folder:

```bash
cd agencies
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```
