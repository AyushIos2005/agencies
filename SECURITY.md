# 🔐 Security Policy

## Supported Versions

We actively support the latest version of this project with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 2.5.x   | :white_check_mark: |
| < 2.5   | :x:                |

---

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

### 📩 How to report

You can report issues by:
- Creating a private issue on GitHub
- Or contacting the developer directly via GitHub:  
  https://github.com/AyushIos2005

---

## ⏱️ Response Time

- Initial response: within 24–72 hours  
- Fix or decision: depends on severity  
- Critical issues: handled immediately

---

## 🔒 What happens next?

- We will review the issue
- Confirm if it is valid
- Fix it in upcoming patch release
- Credit you (if you want)

---

## ❌ What NOT to do

- Do not publicly disclose security issues before reporting
- Do not exploit vulnerabilities

---

## 🙏 Thank You

Thank you for helping keep this project secure.
